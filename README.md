# Simple-Terminal-Assistant V1.0

A simple Java CLI assistant inspired by JARVIS from Iron Man.

## Features

- Startup Banner: shows J.A.R.V.I.S
- Saves user name in name.txt
- Welcomes user back by name
- Command loop
- `help` command
- `hello` command
- `time` command
- `joke` command with random jokes
- `motivate`command with random 3am motivational quotes
- `misson` command with random coding mission/task
- `calc` command for `+`, `-`, `*`, `/`
- Divide by zero check in calculator
- `scan` fake fun command 
- `history` command for current session
- Saves command history to `history.txt` on exit
- `clear` command 
- Text typing animation
-  `settings` menu
- - Toggle typing animation
- - Change animation speed
- - Change user name from settings
- - Clear saved command history
- - Them memu 
- - - None (default)
- - - Red
- - - Green
- - - Blue
- - - Cyberpunk
- Theme stays after clear
- Terminal color reset on exit
- Basic error handling with `try/catch`
